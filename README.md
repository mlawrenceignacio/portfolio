# Mitch Lawrence Portfolio

A responsive full-stack developer portfolio built with React, TypeScript, Tailwind CSS, Supabase, and Vercel.

## Features

- Responsive sidebar-based layout
- Animated hero, profile, projects, skills, and credentials sections
- Backend-driven projects, skills, and certifications using Supabase
- Contact form with message storage and email notifications
- Clean dark UI with green and wine-red accents

## Tech Stack

- React
- TypeScript
- Vite
- Tailwind CSS
- Framer Motion
- Supabase
- Vercel
- Resend

## Deployment

Frontend is deployed on Vercel.  
Data storage and Edge Functions are powered by Supabase.
